# 🧭 Sakinah Compass

> *"Verily, in the remembrance of Allah do hearts find rest."* — Surah Ar-Ra'd (13:28)

A digital mindfulness companion that helps Muslims stay grounded, practice gratitude, and strengthen reliance on Allah (tawakkul).

## 🌟 Features
- Time-aware reflection prompts
- Private journal (Tawfiq Tracker) with localStorage
- Dua & intention reminder
- Mobile-responsive, accessible design

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, gradients, backdrop-filter)
- Vanilla JavaScript
- LocalStorage API
